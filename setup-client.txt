<Tujuan/scope>

Buat UI yang intuitive, modern, dan minimalis dengan react JS, vite dan Tailwind CSS yang sudah dipersiapkan. Dengan desain UI bergaya dark mode cyberpunk

</Tujuan/scope>

<Ketentuan>

Struktu UI nya adalah sebegai berikut:
- Topbar (sebelah kiri menampilkan "Nihonskuy" dengan logonya NS)
- Konten dengan margin kiri kanan.

Di dalam konten ada beberapa input type text:
- Lokasi
- Range Gaji
- Posisi Pekerjaan

Ada dynamic input, bisa ditambahkan atau dikurangi inputnya untuk tujuan membuat point-point:
- Syarat
- Benefit

Jika user sudah mengisi, ada tombol submit yang mentrigger ke backend dengan ketentuan sebagai berikut:
- [POST] http://localhost:8080/api/generate-poster
- Request-example : {
  "lokasi": "SAITAMA - MISATO",
  "gaji": "¥272,200 - ¥302,200",
  "posisi": "RESTORAN (HOTEL)",
  "syarat": [
    "Domisili Jepang",
    "JLPT N4",
    "Kaiwa N3"
  ],
  "benefit": [
    "Kenaikan Gaji",
    "Lembur",

  ]
}
